# Mireru - Lightweight File Viewer for CLI

Mireru is a lightweight file viewer. It can launch from CLI. Powered by Ruby/GTK3.

## Installation

Add this line to your application's Gemfile:

    gem 'mireru'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mireru

## Usage

    $ mireru <file path>
